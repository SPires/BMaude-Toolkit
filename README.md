Orientação: Christiano de Oliveira Braga

Título do Projeto: Refinamento, Abstração e Verificação

Objetivos do projeto:

Este projeto destina-se ao estudo da aplicação de verificação automática (model checking) de propriedades de segurança descritas em lógica temporal linear a especificações descritos segundo o método B. O processo de verificação se dará por meio da semântica da linguagem subjacente à linguagem de design do método B. Essa semântica possui comportamento de autômatos, o que permite a aplicação de verificação de modelos.

<br>
<br>

Este repositório contém:

* Um arquivo com um conjunto de operadores de B e sua semântica formal.

* Um simples e didático conjunto de testes.

<br>
<br>

Utilização:

1- Abra o sistema Maude no terminal.

2- Carregue o arquivo que contém as especificações de B em Maude (BMaude.maude).

3- Escolha um exemplo e abra o arquivo dentro do ambiente Maude.

A especificação e teste de diferentes propriedades podem ser adicionadas ao final de qualquer arquivo de testes. Basta seguir a sintaxe dos comandos utilizados no exemplo.

<br>
<br>

Fundação de Amparo a Pesquisa do Estado do Rio de Janeiro - FAPERJ.

Process: E-26/201.989/2017

Advisor: Christiano de Oliveira Braga

Work Title: Refinement, Abstraction and Verification

Objectives:

The purpose of this work is to study the application of automatic verification of security properties described at linear temporal logic to specs in B method.
The verification process goes through the semantics of the underlying language to the design language from the B method. This semantics behave like automatons, which allows the application of verification models. 

<br>
<br>

Usage:

1- Open Maude system in Terminal

2- Load the file with the N specifications in Maude (BMaude.maude)

3- Choose an example and open the file inside Maude's environment

Specification and test of other properties can be added to the end of any tests file. Just follow the commands syntax used on the example.
